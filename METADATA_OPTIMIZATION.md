## 実装方法

各ページのディレクトリに`layout.tsx`を作成し、`metadata`エクスポートまたは`generateMetadata`関数を使用してメタデータを設定しています。

### 静的メタデータの例

```typescript
// app/about/layout.tsx
import type { Metadata } from "next";

export const metadata: Metadata = {
  title: "会社情報 | マクセラス",
  description: "...",
  openGraph: {
    title: "会社情報 | マクセラス",
    description: "...",
    type: "website",
  },
};
```

### 動的メタデータの例

```typescript
// app/services/[slug]/layout.tsx
import type { Metadata } from "next";
import { getServiceBySlug } from "../../data/services";

export async function generateMetadata({
  params,
}: {
  params: { slug: string };
}): Promise<Metadata> {
  const service = getServiceBySlug(params.slug);
  // ...
}
```

## SEO最適化のポイント

1. **タイトル**: 各ページの内容を明確に示し、「| マクセラス」で統一
2. **ディスクリプション**: 120-160文字程度で、ページの価値提案を簡潔に説明
3. **Open Graph**: SNSシェア時の表示を最適化
4. **キーワード**: 自然に含める（過剰なキーワード詰め込みは避ける）
